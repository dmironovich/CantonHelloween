1. Clone the repo or download ZIP.
2. Add your character image:
   - File name: `character.png`
   - Location: `assets/character.png`
3. Add sound files to `assets/sounds/`:
   - `background.mp3`, `jump.mp3`, `collect.mp3`
4. Open `index.html` in browser or publish via GitHub Pages.
